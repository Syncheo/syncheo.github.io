---

### Plugin — Show Workitem Workflow

IBM EWM does not expose the full workflow of a work item in its editor. To find out which states are available and how to transition between them, contributors had to open the project administration — provided they had the rights.

We developed a plugin that adds a button to the work item editor toolbar. On hover, an interactive state diagram appears in a popup, rendered as SVG directly in the browser with no external network calls.
